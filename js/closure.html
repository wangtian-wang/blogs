闭包的定义 #

函数能记住自己被定义时的作用域,并且访问该作用域的变量,那这个函数就叫做闭包函数

闭包的优点 #

形成块级作用域，减少全局变量污染

封装私有变量

闭包的缺点 #

过度使用闭包 会引起内存泄漏

闭包的应用 #

节流 防抖 柯力化函数 结果缓存 模块化 事件 定时器

闭包函数的 this 指向 #

因为闭包函数被返回到了全局 所以闭包中的 this 指向 window

闭包与词法作用域的区别

嵌套函数 变量查找规则适用于词法作用域

词法作用域: 根据源代码中声明变量的位置来确定变量在何处可以使用

闭包特点: 每个闭包都有一个词法作用域
